### Bonus
* 自由式排版
    * 在home和browser中把矩形图片显示成方形，使用的方法是图片外套div，使用overflow:hidden，调整div的大小为方形并且使图片width大于div
* 响应式布局
    * 没有太多使用技巧或者组件，主要就是各种位置调整都没有写死，尽量使用relative和百分比来调整位置
* 界面美观
    * border都进行了圆角处理，色调做到了尽量统一简洁
